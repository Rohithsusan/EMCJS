🎓 Student Details Form

A simple and interactive Student Management UI built using HTML, Tailwind CSS, and JavaScript.
This project allows users to enter student details and dynamically display them in a table.

🚀 Features
Add student details (Name, Age, Course, Gender, Email)
Dynamic table update using JavaScript
Delete student entries
Basic form validation (prevents empty inputs)
Clean and responsive UI using Tailwind CSS
🛠️ Technologies Used
HTML5
Tailwind CSS
JavaScript (DOM Manipulation)
📂 Project Structure
📁 project-folder
 ├── index.html
 ├── practice.html (optional)
 └── README.md
💡 How It Works
User enters details in the form
Clicks Save
Data is captured using JavaScript
A new row is dynamically added to the table
User can delete any row using the Delete button
🔒 Validation
Prevents empty inputs
Ensures all fields are filled before submission
